def compute_L(B,A):#compute L that gives the Loss function for the given matrix B. L = sum(sum([B-AtA]**2))
    M = np.matmul(np.transpose(A),A)
    temp2 = np.subtract(B,M)
    temp3 = np.matmul(temp2,temp2)
    L = np.sum(temp3)
    return L

def compute_G(B,A):# G(A) = 4At(AtA - B)
    At = np.transpose(A)
    AtA = np.matmul(At,A)
    temp2 = np.subtract(AtA,B)
    return np.dot(4, np.matmul(A,temp2))

Inferences

  • Suppose columns of matrix A are vectors[a1,a2,...,ad] then A^tA is a matrix whose elements contain the cross product of each of the row vectors of matrix A. It gives the relation between the row vectors of matrix A , their magnitude and their cosines as well.
  • Thus when we tried to start from different initial matrix A and converge to one A, it wasnt possible because the magnitude were totally different and angles between the row vectors of matrix A as well.
  • Same goes for a the different k x D matrix we generated in the previous example.
  • In all these cases we are trying to optimize a value A^tA and not the matrix A. There could be multiple matrices A with different k as dimension (k x D), that give the same A^tA. It is possible because it is just the different combination of mulitple dot products of row vectors of matrix A.
